# 学生成绩管理系统 python181

本项目是基于Python语言，采用Django框架开发的学生成绩管理系统，使用MySQL数据库进行数据存储。

## 技术栈

- Python
- Django
- MySQL（建议使用版本8）

## 功能模块

系统包含以下主要功能模块：

### 账号注册

- 学生账号
- 老师账号
- 管理员账号

### 学生功能

- 修改个人信息
- 查看课表
- 查看成绩
- 修改密码

### 老师功能

- 修改个人信息
- 查看课程
- 录入学生成绩
- 修改密码

### 管理员功能

- 添加系
- 修改系信息
- 添加班级，修改班级，修改班级课表
- 添加课程，修改课程信息
- 添加学生账号
- 添加老师账号
- 修改学生信息
- 修改老师信息

## 系统角色

- 学生
- 老师
- 管理员
